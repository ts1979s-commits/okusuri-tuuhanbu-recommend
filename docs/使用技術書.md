# 使用技術書 - お薬通販部商品レコメンドAI

## 🛠️ **技術スタック概要**

### システム構成
```
Frontend: Streamlit (Python Web Framework)
    ↓
Backend: Python + pandas (Data Processing)
    ↓  
Search Engine: 文字列マッチング（メイン）+ FAISS/AI（オプション/拡張用）
    ↓
Data Layer: CSV（product_recommend.csv）
    ↓
Deployment: Streamlit Cloud + GitHub (CI/CD)
```
> ※ FAISS/AIベクトル検索は現状未使用・拡張用サンプルです。

---

## 📋 **主要技術詳細**

### 1. フロントエンド技術

#### **Streamlit v1.28+**
- 迅速なWebアプリ開発、Python単一言語、豊富なUIコンポーネント、デプロイ容易
- サイドバー・検索UI・結果表示・レスポンシブ対応

---

### 2. バックエンド技術

#### **Python 3.11+**
- データ処理・ファイルI/O・例外処理・OOP

#### **pandas v2.0+**
- CSVデータの効率的処理、強力な検索・フィルタ機能、重複除去

---

### 3. 検索エンジン技術

#### **文字列マッチング（メイン）**
- pandasのstr.containsや==による部分一致・完全一致検索

#### **FAISS/AI（オプション/拡張用）**
- FAISS + OpenAI Embeddingsによるベクトル検索（現状未使用・拡張候補）
- 利用時はrequirements.txt・app.pyで有効化

---

### 4. データストレージ技術

#### **CSV (product_recommend.csv)**
- メイン商品データベース、UTF-8、バージョン管理容易

#### **Pickle/JSON（FAISS用キャッシュ/設定）**
- FAISS/AI拡張時のみ利用

---

### 5. デプロイメント技術

#### **Streamlit Cloud**
- GitHub連携による自動デプロイ、無料プラン利用

#### **GitHub CI/CD**
- mainブランチpushで自動ビルド・デプロイ

---

### 6. 開発・運用ツール

- Python仮想環境（venv）
- Visual Studio Code（推奨拡張あり）
- Git + GitHub（バージョン管理）
- requirements.txt（依存管理）

---

## 📊 **パフォーマンス・スケーラビリティ**

- 35商品規模でデータロード・検索は高速（<100ms）
- 拡張時はDB移行・インデックス最適化等で対応可能

- 入力検証・エラーハンドリング・ログ出力
