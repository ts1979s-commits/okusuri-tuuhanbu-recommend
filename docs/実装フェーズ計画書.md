# 実装フェーズ計画書 - お薬通販部商品レコメンドAI

## 📋 フェーズ構成と方針

本プロジェクトは「Web UI（Streamlit）＋pandasによる基本検索・UI・データ処理」を中心に、段階的な拡張を想定しています。

- 必須フェーズ：Web UI・基本検索・データ整備・デプロイ
- 拡張フェーズ：FAISS/AIベクトル検索、API化、ML強化、スケーリング等（今後の候補）
- 各フェーズの成果物・完了基準を明記

---

## 🟩 必須フェーズ

### Phase 1: 基盤構築・環境整備
- Python/Streamlit環境構築、GitHubリポジトリ作成、データ設計
- 成果物：開発環境、product_recommend.csv、初期プロジェクト構造

### Phase 2: 基本検索・UI実装
- 症状・商品名・カテゴリ検索、UI/UX、重複除去、エラーハンドリング
- 成果物：app.py、検索UI・ロジック、重複除去機能

### Phase 3: データ整備・デプロイ
- product_recommend.csv整備、Streamlit Cloudデプロイ、README/技術文書作成
- 成果物：最新データ、デプロイ済みWebアプリ、README.md、技術文書

## 🟦 拡張フェーズ（今後の候補）

- 成果物：拡張済みインフラ、運用ドキュメント
